## Basic Installation
```bash
pip install agent4all
```

## Framework-specific Installation

### Agent4ALL Agents Support (Default)

```bash
pip install agent4all
``` 

### CrewAI Support
```bash
pip install "agent4all[crewai]"
```

Includes:
- CrewAI framework
- Agent4ALL tools integration
- Task delegation capabilities
- Sequential and parallel task execution

### AG2 Support
```bash
pip install "agent4all[autogen]"
```

Includes:
- AG2 framework
- Agent4ALL tools integration
- Multi-agent conversation capabilities
- Code execution environment

### Both Frameworks
```bash
pip install "agent4all[crewai,autogen]"
```

## Additional Features

```bash
# UI support
pip install "agent4all[ui]"

# Chat interface
pip install "agent4all[chat]"

# Code interface
pip install "agent4all[code]"

# Realtime voice interaction
pip install "agent4all[realtime]"

# Call feature
pip install "agent4all[call]"
```
