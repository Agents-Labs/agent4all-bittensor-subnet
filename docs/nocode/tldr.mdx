## Quick Start with Agent4ALL Agents

```bash
# Install with CrewAI support
pip install agent4all

# Set your API key
export OPENAI_API_KEY="Enter your API key"

# Initialize and run
agent4all --init "create a movie script about dog in moon"
agent4all
```

## Quick Start with CrewAI

```bash
# Install with CrewAI support
pip install "agent4all[crewai]"

# Set your API key
export OPENAI_API_KEY="Enter your API key"

# Initialize and run
agent4all --framework crewai --init "create a movie script about dog in moon"
agent4all --framework crewai
```

## Quick Start with AG2

```bash
# Install with AG2 support
pip install "agent4all[autogen]"

# Set your API key
export OPENAI_API_KEY="Enter your API key"

# Initialize and run
agent4all --framework autogen --init "create a movie script about dog in moon"
agent4all --framework autogen
```

## User Interface

```bash
# Install UI support
pip install "agent4all[ui]"

# Set up environment
export OPENAI_API_KEY="Enter your API key"
chainlit create-secret
export CHAINLIT_AUTH_SECRET=xxxxxxxx

# Run UI
agent4all ui
```
