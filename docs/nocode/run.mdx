---
title: "Run"
description: "Instructions for running Agent4ALL from the command line, including framework selection and automatic mode"
icon: "play"
---

```bash
agent4all
```

or 
    
```bash
python -m agent4all
```

## Specify the agent framework (Optional):

```bash
agent4all --framework autogen
```

## Full Automatic Mode

```bash
agent4all --auto create a movie script about Dog in Moon
```
