## Development Setup

<Steps>
  <Step title="Clone Repository">
    ```bash
    git clone https://github.com/MervinPraison/Agent4ALL.git
    cd src/agent4all-ts
    ```
  </Step>

  <Step title="Install Dependencies">
    ```bash
    npm install
    ```
  </Step>

  <Step title="Build Package">
    ```bash
    npm run build
    ```
  </Step>
</Steps>
